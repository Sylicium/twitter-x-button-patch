Twitter X Button Patch

So does this happen to you too? That weird X button not working as expected to close the page?
No worries, here is the solution!

This simple extension solves this problem by adding simple code to close the page when you now click that X button.
Simple to install, no additional action required. Relax and be happy!

How to install ?

(Note that this extension is only available on Chrome fo now)

1. Right clic on the .crx file in the releases, then Save as and save it.
2. Go to the chrome://extensions/ list page
3. Open a file explorer to the .crx file, and Drag&drop it on the page.

Clic install on the popup and its done !
